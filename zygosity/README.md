### Figure 6: Genotype Count Heatmaps
This script generates genotype count heatmaps (zygosity per variant/gene) for publication figures.

**Inputs:**
- `f6_var.txt`
- `f6_gene_hethom.txt`
- `f6_gene_uhet_hom.txt`

**Outputs:**
- `paper_fig.png`
- `papper.png`

**Dependencies:**
- R packages: ggplot2, dplyr, tidyr, ggpubr, grid
